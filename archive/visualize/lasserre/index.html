The Original Problem

Imagine you have a fixed amount of paint to spread across 3 bins. For every way you spread it, we measure the worst peak across all possible windows. Your goal: find the spread that makes the worst peak as small as possible. That minimum is val(d).

What Is Relaxation?

The original problem is hard because the feasible set (the simplex) is complicated. Relaxation means enlarging the allowed region to make the problem easier to solve. The trade-off: a bigger region gives a weaker bound, but is computationally tractable.

3

Moment Matrices & PSD

A moment matrix is a consistency check on measurements. If the matrix is positive semidefinite (PSD), the measurements could come from a real distribution. If not, they're impossible — no real distribution could produce them.

The Lasserre Ladder

Each order of the Lasserre hierarchy adds stronger consistency checks. Order 1 checks averages. Order 2 checks pairwise products. Order 3 checks triples. Each order rules out more fake solutions, tightening the bound toward the true answer.

The Farkas Certificate

To prove a lower bound, we use binary search. At each threshold t, the SDP checks: "could any distribution (even a fake one) achieve a peak below t?" If the answer is no — the system is infeasible — then the true answer must be above t.
