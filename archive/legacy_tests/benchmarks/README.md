# benchmarks/

Generic SDP / cascade benchmarks (archived).

- `sdp_solvers/` — head-to-head solver comparisons (Mosek/SCS/etc.).
- `profiling/` — bottleneck and high-d profiling scripts.
- `matlab/` — MATLAB benches for SDPNAL / SDPNAL+ install + solve.
- `single_runs/` — minimal one-config drivers (cvxpy / scs / generic).
- `bench_general/` — sweep, scan, bisection, d16-config benches that
  do not fit the above buckets.
